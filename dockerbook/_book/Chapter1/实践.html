实践

初步实践

docker run hello-world

启动一个容器（即从镜像创建一个实例）,docker会自动检测宿主机中是否存在hello-world镜像，如果没有docker会自动从docker仓库中找寻，如果找到会执行docker pull,下载镜像到当前机器。

docker images

查看机器中所有的镜像

docker ps -a

查看所有的容器

docker ps

查看所有运行中的容器
